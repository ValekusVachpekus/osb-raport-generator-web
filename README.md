# Конструктор рапорта ОСБ

Веб-приложение для генерации BB-кода рапорта о дисциплинарном проступке (для публикации на форуме ГАИ Нижегородской области).

🌐 **Сайт:** https://valekusvachpekus.github.io/osb-raport-generator-web/

---

## Возможности

- **Редактор** — заполнение всех полей рапорта: тема, основание, требования, приложения (фото/видео + USB-Flash)
- **Предпросмотр** — читаемый вид рапорта перед публикацией
- **BB-код** — готовый код для вставки на форум, с кнопкой копирования
- **Автосохранение черновика** — данные формы сохраняются в браузере и восстанавливаются при перезагрузке
- **Сохранение настроек** — должность, ФИО и подпись сохраняются между сессиями
- **Индикатор несохранённых изменений** в блоке настроек
- **Валидация** — подсветка незаполненных обязательных полей
- **Динамические вложения** — добавление и удаление ссылок на доказательства
- Адаптивный дизайн — работает на мобильных устройствах
- Тёмная тема

## Использование

Откройте сайт → заполните поля → нажмите **⚡ Сгенерировать** → скопируйте BB-код или просмотрите предпросмотр.

## Локальный запуск

Просто откройте `index.html` в браузере — никаких зависимостей не требуется.

## Технологии

Чистый HTML + CSS + JavaScript. Без фреймворков, без сборки.
